⚠️  KNOWN LIMITATIONS & WORKAROUNDS
═══════════════════════════════════════════════════════════════════════════════

Limitation 1: ARIMA requires statsmodels
├─ Impact: Optional feature
└─ Workaround: pip install statsmodels

Limitation 2: Gemini requires API key
├─ Impact: AI classification skipped
└─ Workaround: Use rule-based fallback

Limitation 3: Large datasets memory intensive
├─ Impact: Monte Carlo performance
└─ Workaround: Reduce simulation count

Limitation 4: Async agents need event loop
├─ Impact: Notebook environment
└─ Workaround: Use wrapper functions
